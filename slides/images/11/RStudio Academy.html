Import and visualize

Welcome to your first project milestone! For the first five weeks of this course, you will investigate actual sales on a variety of products at a variety of stores, provided by Walmart to the M5 forecasting competition for public use. You can find a brief description of the data and its variables in this data dictionary or in the data/walmart_dictionary.Rmd file in your milestone project.

We’ll split this project into five milestones that build on each other to make a complete report of the analysis.

Each milestone will require you to read and edit an R Markdown document, like the one to the right. To learn how to use R Markdown, watch this video.

For each milestone, you will need to:

  1. Recreate a milestone artifact to demonstrate that you’ve mastered a new set of R skills

  2. Go beyond the milestone to demonstrate that you are learning how to teach yourself R. This is the true competency that will make you a lifelong R user. At the weekly project session, you will share your screen and show your groupmates how you have extended the milestone and how they could do the same. Extend and personalize the milestone in whichever way you like. Be curious! Be ambitious!

Follow the instructions in the file kellogg_01_visualization.Rmd to complete this milestone.
